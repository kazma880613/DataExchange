文件檔說明 :

1. Account.txt ... 物件檔 :
	此類型檔案透過SQL語法撈取特定物件資料，內容格式為 JSON ，並包含2項資訊；
	(1).objectItem : 此物件為何
	(2).QueryString : SQL撈取資料的語法；第一句為主資料，第2及第3等句為配合第1句內資料的其餘附加組合，並透過第幾欄位與第幾欄位(下語法時的順序做等式 Ex.1=1)相等進行組合

2. SQLInfo.txt ... SQL連線檔 :
	此檔案用以進行SQL連線的相關設定，目前已 ODBC 連線為主，爾後可擴充針對各類型資料庫更新

3. TodoObject.txt ... 執行順序檔 :
	此檔案目的為將物件檔做當日執行順序的編排，當進行至最後一項時會重製檔案內所有資訊，使用者在此檔案為客戶對傳輸的物件做序列排程

4. apiURL.txt ... API連線檔 :
	此檔案用以將包裝好得資料傳送到何處(URL)
